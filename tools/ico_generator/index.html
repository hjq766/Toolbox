点击或拖拽图片到此处上传

支持PNG、JPG、WebP格式，推荐使用PNG格式

Windows ICO
macOS ICNS

功能介绍

主要功能

  • 支持生成Windows ICO和macOS ICNS格式图标
  • 可生成网站Favicon（16×16、32×32）
  • 自动生成多种尺寸的图标，满足不同场景需求
  • 支持PNG、JPG、WebP等格式转换
  • 支持单个下载和批量打包下载
  • 所有处理在浏览器中完成，保护隐私安全

使用说明

  • 将图片拖拽到上传区域，或点击选择文件
  • 选择目标系统（Windows或macOS）
  • 选择需要的图标尺寸（可多选）
  • 点击"开始生成"进行转换
  • 可以单独下载每个尺寸的图标，或使用批量下载

使用场景

  • 网站Favicon：选择16×16或32×32尺寸，可用于网站标签页图标
  • 桌面应用：选择多个尺寸生成应用程序图标
  • 开发项目：批量生成不同尺寸的图标资源

注意事项

  • 建议上传尺寸较大的图片（至少512×512）以确保生成的小尺寸图标清晰
  • 推荐使用PNG格式的图片作为源文件，以获得最佳效果
  • 如果源图片太大，建议先进行压缩
  • 生成的图标会自动调整为正方形，保持图像比例并居中显示
  • 制作Favicon时，图案尽量简单清晰，避免复杂细节
