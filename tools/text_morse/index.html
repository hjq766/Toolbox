摩尔斯电码转换器

支持中英文、数字与摩尔斯电码的双向转换，快速准确

文本内容

摩尔斯电码

功能说明

摩尔斯电码转换器支持将中英文、数字等文本转换为摩尔斯电码，也可以将摩尔斯电码转换回原文本。由于摩尔斯电码本身只支持英文字母，中文字符会先转换为拼音后再进行编码。

使用说明

  • 文本转摩尔斯电码：
    • 英文和数字：直接转换为对应的摩尔斯电码
    • 中文字符：先转换为拼音，再将拼音转换为摩尔斯电码
    • 例如："你好" → "ni hao" → "-. .. / .... .- --- /"
  • 摩尔斯电码转文本：
    • 英文和数字：可以完整还原原文
    • 中文内容：只能还原为拼音形式，无法还原为原中文字符
  • 摩尔斯电码规则：
    • 使用 "." 表示点，"-" 表示划
    • 字符之间用空格分隔
    • 单词之间用 "/" 分隔

支持格式

  • 英文字母（A-Z，不区分大小写）：可完整互转
  • 数字（0-9）：可完整互转
  • 常用标点符号：可完整互转
  • 中文字符：转换为拼音后编码，无法还原为原中文
