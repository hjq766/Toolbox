关于工具

URL编码转换工具支持将URL地址转换为十六进制编码格式，也可以将编码内容解码为原始URL。

主要特点

  • 支持批量转换，一次处理多个URL
  • 实时字符统计，直观显示内容长度
  • 智能识别URL协议，自动保留http/https前缀
  • 支持快捷键操作，提高使用效率
  • 可导出转换结果，支持多种格式

快捷键

Ctrl + Enter 快速编码
Ctrl + Shift + Enter 快速解码
Ctrl + C 复制结果
