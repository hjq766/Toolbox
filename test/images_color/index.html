工具说明

什么是图片色板生成？

图片色板生成是一种从图片中分析并提取主要颜色的技术，可以帮助设计师快速获取图片中的配色方案，用于设计创作和色彩搭配。

主要功能说明

自动提取：上传图片后自动分析并提取主要颜色，可设置提取数量。

选区取色：支持框选图片特定区域，提取该区域的主要颜色。

颜色排序：可按显著度、亮度、色相等方式对颜色进行排序。

导出功能：支持导出色板图片，可选择是否显示颜色值。

使用步骤

1. 点击或拖拽图片到上传区域

2. 等待自动提取主要颜色

3. 可使用选区工具提取特定区域颜色

4. 点击颜色卡片可复制颜色值

5. 可导出色板图片保存配色方案

使用建议

1. 上传清晰的图片以获得更准确的颜色提取结果

2. 使用选区工具可以更精确地提取特定区域的颜色

3. 尝试不同的颜色数量和排序方式以获得最佳效果

4. 建议图片大小不超过 5MB，支持 JPG、PNG、GIF 等常见格式
