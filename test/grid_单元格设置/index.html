功能介绍

CSS Grid 生成器是一个可视化的网格布局工具，支持智能布局推荐和实时预览。

主要功能

  • 智能布局推荐，快速生成最佳网格方案
  • 可视化调整网格行列数和间距
  • 支持单元格合并和调整
  • 实时生成 CSS 代码
  • 响应式预览效果
  • 支持导出预览图

使用说明

  • 设置容器宽度和期望列数，获取智能布局推荐
  • 使用网格设置调整具体的行列数和间距
  • 点击单元格可以调整其跨度
  • 实时查看生成的 CSS 代码
  • 可以切换不同设备尺寸预览效果
