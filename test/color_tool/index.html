功能简介

                    
颜色工具集提供全面的颜色相关功能，帮助设计师和开发者更好地处理颜色。

                    
                    
主要功能

                    
    
                        
  • 支持多种颜色格式转换（HEX、RGB、RGBA、HSL、HSB、CMYK）
    • 
                        
  • 生成颜色的系统梯度（0-900）
    • 
                        
  • 智能生成状态颜色（Success、Info、Warning、Danger）
    • 
                        
  • 提供多种配色方案（互补色、类似色、三分色等）
    • 
                        
  • 所有颜色值支持一键复制
    • 
                    

                    
                    
使用说明

                    
    
                        
  • 使用颜色选择器或直接输入颜色值
    • 
                        
  • 支持多种格式的颜色值输入（HEX、RGB、HSL等）
    • 
                        
  • 点击任意颜色值可以快速复制
    • 
                        
  • 色阶和配色方案会根据选择的颜色自动更新
